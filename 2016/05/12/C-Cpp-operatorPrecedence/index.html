C/C++ 操作符优先级

发表于 2016-05-12   |   分类于 C/C++

参考

  • cppreference: C++ Operator Precedence
  • C语言中文网：C语言运算符的优先级和结合性一览表

优先级

  1. 单目运算符，只需一个操作数，先右后左，越靠近操作数越优先，如：++*p.f 即 ++(*(p.f))
  2. 算术运算符：+, -, *, /, %
  3. 移位运算符：>>, <<
  4. 关系运算符：>, <, == 等
  5. 位逻辑运算符: &, |, ^
  6. 逻辑运算符：&&, ||
  7. 条件运算符：先右
  8. 赋值运算符：先右，如：a=b=b+c 即 a=(b=b+c)
  9. 逗号: 自左向右运算，仅保留最后一个值

其实没有必要记下来，如果不确定就加括号。 ：P
