Ubuntu下SoX音频处理工具

发表于 2017-07-21   |   分类于 Tools

SoX是Ubuntu下一个音频处理工具，运行于命令行，安装如下。SoX包含三条主要命令
sox、play和rec，用前者可实现SoX所有功能，而后两者只是为了简化“播放”和“录音”操作。另一条命令soxi可输出音频信息。SoX支持一些常见的音频格式，如wav/mp3/ogg等，参见手册。
